⚡ Interactive Tactical Portfolio 

A high-performance, interactive personal portfolio featuring a dark "stealth" aesthetic, CSS-rendered 3D hardware, and dynamic glassmorphism carousels.

🛸 Overview

This repository contains my personal portfolio website, designed to break away from traditional resume formats. It blends clean, modern layout principles with aviation/sci-fi cockpit aesthetics to create an immersive browsing experience. 

The site is built completely without heavy UI libraries, relying on advanced CSS techniques and Vanilla JavaScript to achieve 3D rendering, complex shadows, and dynamic background synchronization.

✨ Key Features

* Tactical Telemetry UI: A custom-built grid layout mimicking a physical hardware dashboard with matte black panels and machined edge-lighting.
* Hyperrealistic CSS Hardware: Floating 3D ambient details (glowing LEDs, metallic toggle switches, hazard decals) rendered entirely in CSS using layered box-shadows and radial gradients.
* Dynamic Glassmorphism Carousels: Education and Experience timelines built as swipeable glass cards (`backdrop-filter: blur`) over dynamic backgrounds.
* Synchronized Parallax: Custom JavaScript synchronizes the background image of the section to match the active carousel card, complete with a desktop parallax effect.
* Responsive & Stable: Complex CSS geometries and parallax effects are carefully scoped with media queries to ensure a flawless, standard-scrolling experience on mobile devices.

🛠️ Tech Stack

* HTML5: Semantic structure and accessible layout.
* CSS3: Advanced styling including `clip-path`, `linear-gradient` shading, layered `box-shadow` for 3D depth, and CSS animations.
* Vanilla JavaScript (ES6): Lightweight DOM manipulation for the custom carousel logic and background synchronization. No external libraries or dependencies.
